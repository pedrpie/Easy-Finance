# Easy Finance

Uma aplicação full-stack de controle financeiro desenvolvida para gerenciar receitas, despesas e visualizar informações financeiras de forma simples e organizada.

Este projeto foi criado como prática de desenvolvimento moderno utilizando Next.js no frontend e backend, com TypeScript para tipagem segura e componentes estilizados com TailwindCSS e shadcn/ui.

## Tecnologias utilizadas
### Frontend

- React

- Next.js

- TypeScript

- TailwindCSS

- shadcn/ui

- Lucide Icons

### Backend

- API Routes do Next.js

- Middleware

- Comunicação via HTTP (fetch)

## Conceitos aplicados

Arquitetura full-stack com Next.js

Componentização de UI

Criação de APIs REST

Comunicação entre frontend e backend

Tipagem com TypeScript

Organização de projeto escalável

Estilização com TailwindCSS

Uso de bibliotecas de componentes (shadcn/ui)

## Como rodar o projeto
```bash
# Clone o repositório:
git clone github.com/pedrpie/Easy-Finance.git

# Entre na pasta do projeto:
cd easy-finance

# Instale as dependências:
npm install

# Rode o servidor de desenvolvimento:
npm run dev

# O projeto estará disponível em:
http://localhost:3000
````
